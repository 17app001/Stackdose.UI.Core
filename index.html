設計器工具

類似 Unity Scene Editor 的拖曳設計器，讓設計師直接拖曳 PLC 控制項、設定地址、輸出 JSON； 工程師用 app-config.json 設定 PLC 連線即可上線， 整合流程從數月縮短至數天，全程符合 FDA 21 CFR Part 11 稽核要求。

快速上手 — 設計師工作流程
Step 1 — 開啟設計器
  1. 開啟方案 Stackdose.Designer.sln
  2. 啟動 MachinePageDesigner（預設 Startup Project）
  3. Ctrl+N 新建，或 Ctrl+O 開啟現有 .machinedesign.json
Step 2 — 拖放設計
  1. Toolbar「📌 Tags」先定義 PLC 地址清單（D100=溫度…）
  2. 從左側 Toolbox 拖曳控制項至畫布
  3. 右側 Properties 面板 — PLC 位址欄可下拉選取 Tags，或直接輸入
  4. 右鍵選單：複製、刪除、Z-Order、鎖定
  5. Ctrl+S 儲存為 .machinedesign.json
Step 3 — 預覽與驗證
  1. DesignViewer：拖入 JSON 靜態預覽外觀
  2. DesignRuntime：輸入 PLC IP → 連線 → 載入 JSON → 觀察即時數值
  3. 儲存後畫面自動熱更新（400ms 延遲）
Step 4 — 量產部署
  1. 將 DesignPlayer Release 發布到現場機台
  2. 編輯 Config/app-config.json（PLC IP、畫布路徑）
  3. 或啟動後到 Settings 頁面直接修改，不需重新部署
