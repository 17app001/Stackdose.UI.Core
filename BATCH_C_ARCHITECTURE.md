## ?? 生命週期流程圖

### **CyberControlBase 生命週期**

```
┌─────────────────────────────────────────────────────────────┐
│                        建構函數                                │
│                  (Constructor)                                │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ 1. 檢測設計模式 (_isInDesignMode)                       │   │
│  │ 2. 註冊 Loaded/Unloaded 事件                           │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                     Loaded 事件觸發                            │
│          (CyberControlBase_Loaded)                            │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ 1. 檢查設計模式？ → 是：跳過                            │   │
│  │ 2. 註冊到 ThemeManager (如實作 IThemeAware)           │   │
│  │ 3. 呼叫 OnControlLoaded() ← 子類覆寫                  │   │
│  │ 4. 設定 _isLoaded = true                              │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      控件運作中                               │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ - 主題變更通知 (OnThemeChanged)                        │   │
│  │ - UI 操作 (SafeInvoke / SafeBeginInvoke)              │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                    Unloaded 事件觸發                           │
│         (CyberControlBase_Unloaded)                           │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ 1. 呼叫 OnControlUnloaded() ← 子類覆寫                │   │
│  │ 2. 從 ThemeManager 註銷                               │   │
│  │ 3. 設定 _isLoaded = false                             │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                     Dispose (可選)                            │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ 1. 移除事件訂閱                                         │   │
│  │ 2. 從 ThemeManager 註銷                               │   │
│  │ 3. 設定 _disposed = true                              │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

│  ┌──────────────────────────────────────────────────────┐   │
│  │ 1. 訂閱 ConnectionEstablished 事件                     │   │

│  │ )                                                     │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                PLC 數據更新 (週期性觸發)                       │
│           (OnPlcScanUpdated)                                  │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ SafeInvoke(() =>                                      │   │

└─────────────────────────────────────────────────────────────┘

## ?? 設計原則

### **SOLID 原則應用**

1. **Single Responsibility (單一職責)**
   - ? CyberControlBase：只負責通用控件邏輯
   - ? PlcControlBase：只負責 PLC 相關邏輯
   - ? 具體控件：只負責自己的業務邏輯

2. **Open/Closed (開放封閉)**
   - ? 基類對擴展開放（子類可覆寫虛方法）
   - ? 基類對修改封閉（不需修改基類就能新增功能）

3. **Liskov Substitution (里氏替換)**
   - ? 子類可以完全替換父類
   - ? 覆寫方法不改變基類行為

4. **Interface Segregation (介面隔離)**
   - ? IThemeAware：只定義主題相關方法
   - ? IDisposable：只定義清理相關方法

5. **Dependency Inversion (依賴反轉)**
   - ? 依賴抽象（IThemeAware、IPlcManager）
   - ? 不依賴具體實作
