# 操作員手冊 — DesignPlayer 使用指南

> **適用對象：** 機台操作員（現場作業人員）  
> **版本：** 2026-04  
> **相關系統：** Stackdose DesignPlayer

---

## 1. 系統概述

DesignPlayer 是機台監控介面，顯示 PLC 即時數值，並提供受權限管控的操作按鈕。系統符合 **FDA 21 CFR Part 11** 合規要求，所有操作均會記錄到稽核日誌。

---

## 2. 登入與登出

### 2.1 登入

1. 啟動 DesignPlayer，若系統啟用登入管控，會顯示登入對話框。
2. 輸入**帳號**與**密碼**，按「登入」。
3. 登入成功後進入主監控畫面。

> **注意：** 每次登入都會記錄到稽核日誌，包含帳號、時間、IP。

### 2.2 登出

- 點選左側導航列底部的「登出」按鈕，或點選頁首右側帳號名稱 → 「登出」。
- 登出後系統回到登入畫面（若已啟用登入管控）。

### 3.1 頁首資訊

| 顯示項目 | 說明 |

| 機台名稱 | 設備識別 ID（由工程師設定） |
| PLC 連線狀態 | 🟢 已連線 / 🔴 斷線 / 🟡 重連中 |

### 3.2 頁籤列（多頁面）

若設計師設計了多個頁面（如「主頁」、「警報」、「日誌」），可點選頁籤切換。單頁設計時頁籤列會自動隱藏。

## 4. 讀取 PLC 數值

### 4.1 數值顯示（PlcLabel）

顯示 PLC 寄存器的即時數值，例如溫度、壓力、計數器。

- 數值每 500ms 自動更新
- 若 PLC 斷線，顯示最後已知值（或 `—`）

### 4.2 狀態指示燈（PlcStatusIndicator）

- 🟢 **亮起（綠/自訂顏色）**：PLC 位元 = 1（ON）
- ⚪ **熄滅**：PLC 位元 = 0（OFF）

---

## 5. 操作按鈕（SecuredButton）

受權限管控的按鈕，點擊時依設定執行不同動作：

| 按鈕模式 | 行為 |
|---|---|
| `write` | 寫入指定值到 PLC 位址 |
| `pulse` | 寫入值 → 等待指定毫秒 → 寫 0（例如觸發啟動脈衝） |
| `toggle` | 讀取當前值並翻轉（ON→OFF 或 OFF→ON） |
| `sequence` | 執行多步驟 PLC 指令序列（由工程師設計） |

### 5.1 操作流程

1. 點擊按鈕
2. **若需要驗證**：輸入密碼確認或選擇授權使用者
3. 確認後系統執行對應 PLC 寫入
4. 操作結果記錄到稽核日誌（帳號 + 時間 + 位址 + 寫入值）

> **重要：** 每次 SecuredButton 操作均記錄稽核日誌，包含失敗嘗試。請勿嘗試使用他人帳號操作。

---

## 6. 警報查看（AlarmViewer）

### 6.1 查看現有警報

警報列表顯示目前 PLC 中有效的警報狀態：

| 欄位 | 說明 |
|---|---|
| 群組 | 警報分類（如「馬達」、「加熱器」） |
| 說明 | 警報描述（如「馬達過載」） |
| 狀態 | 🔴 觸發中 / ⚪ 正常 |

### 6.2 警報觸發

- 警報觸發時，畫面上的警報列表該列會變色標示。
- 排除故障後，對應的 PLC 位元歸零，警報自動清除。

---

## 7. 感測器數值查看（SensorViewer）

SensorViewer 顯示組合式感測器狀態（多個 PLC 位元的 AND / OR 判斷結果）：

| 欄位 | 說明 |
|---|---|
| 群組 | 感測器分組 |
| 說明 | 感測器狀態說明 |
| 狀態 | ✅ 正常 / ⚠ 異常 |

---

## 8. 即時日誌查看（LiveLog）

若頁面中有 LiveLog 面板，可查看系統即時訊息：

- 藍色：一般資訊
- 橘色：警告
- 紅色：錯誤 / 警報

---

## 9. 日誌查詢（LogManagement，需 Operator 以上權限）

1. 從左側導航進入「日誌管理」。
2. 選擇日誌類型：操作日誌 / 事件日誌 / 週期數據。
3. 設定時間範圍，點「查詢」。
4. 可匯出為 PDF（「匯出」按鈕）。

---

## 10. PLC 連線狀態處理

| 狀態 | 顯示 | 說明 |
|---|---|---|
| 正常 | 🟢 已連線 | PLC 數值持續更新 |
| 斷線 | 🔴 斷線 | 系統自動嘗試重連 |
| 重連中 | 🟡 重連中... | 等待系統自動重新連線 |

> **一般操作員無需手動操作。** 若斷線後長時間無法重連，請聯絡工程師。

---

## 11. 常見問題

**Q: 按下按鈕沒有反應？**
A: 可能原因：(1) 權限不足 — 請確認您的角色是否有權執行此操作；(2) PLC 斷線 — 等待重連後再試。

**Q: PLC 數值不更新？**
A: 確認頁首 PLC 狀態為 🟢 已連線。若顯示 🔴 斷線，請等待自動重連或聯絡工程師。

**Q: 數值顯示為 `—` 或空白？**
A: PLC 斷線時顯示最後已知值或空白。重連後自動恢復。

**Q: 忘記密碼？**
A: 請聯絡系統管理員（Admin 角色）重設密碼。

---

## 12. 稽核日誌說明（FDA 21 CFR Part 11）

本系統符合 FDA 21 CFR Part 11 電子記錄要求：

- **所有登入/登出**均記錄
- **所有 SecuredButton 操作**均記錄（包含失敗嘗試）
- **系統事件**（啟動、PLC 連線/斷線）均記錄
- 記錄不可刪除，只能查詢與匯出

> 請使用您自己的帳號操作。與帳號對應的所有操作記錄具有法律效力。
